Sudoku Game Rules

Developed by Aifaz khan

Objective:

Fill a 9x9 grid with digits from 1 to 9, ensuring each column, row, and 3x3 subgrid contains all digits.

Points System:

Players earn points based on time and lose points for errors.

  • Time Points: 1 point deducted for each second elapsed.
  • Error Points: 50 points deducted for each incorrect entry.

Winning Criteria:

The player with the higher total points at the end is the winner.

Game Completion:

Complete the game by filling the entire 9x9 grid with valid digits.

Scoring Example:

If a player completes the game in 5 minutes with 2 errors:
Time Points: 300 seconds * 1 point/second = 300 points
Error Points: 2 errors * 50 points/error = 100 points
Total Points = Time Points - Error Points = 300 - 100 = 200 points.

Winner Declaration:

The winner will be announces shortly after the event completion

Game Restart:

Players cannot restart the game once it is started or he/she will be declared disqualified.
